# Dark Magic – FiveM

Welcome to the script magic test beta 1.1

## Description
FiveM dark magic script.  
A black energy comes out of the player’s hand and burns the targeted area for a short time.

## Commands
- /darkmagic  
  Enable / disable dark magic

- /darkmagicoff  
  Force disable dark magic

## How to use
1. Use /darkmagic  
2. Aim with your camera  
3. Hold **E**  
4. Dark magic is cast from the hand  
5. Effects stop automatically after a few seconds

## Auto disable
- On player death  
- When switching to unarmed
